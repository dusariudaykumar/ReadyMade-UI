Avatar


The Avatar component is used to represent a user, and displays the profile picture, initials .



Image Avatar:

Avatar is available in 4 different sizes. You can use image in Avatar.

To use avatars of different sizes, add any one of the size classs along with the base class- avatar-xs, avatar-xs-size, avatar-sm-size, avatar-md-size, and avatar-lg-size. By default a medium sized, circle avatar is generated.

Letter avatars

The Letter avatar available in 3 different sizes. To get a text avatar use the base class avatar-text along with the base class
